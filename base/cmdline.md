# 命令终端提示符号：
---

| [root@teacher ~]#  | 命令行终端提示符 
| --------           | :----
| root               |代表的是当前登陆用户的用户名，这里是管理员root
| @                  |分隔符号
|teacher             |是当前linux主机的主机名，这里是teacher
|~                   |代表是当前用户所在的路径，这里是root的家目录
|[ ]                 |是分隔符号
|#                   | 用户提示，这里是超级管理员的提示符  （$是普通用户）
